# 闭包

简单闭包：

```rust
fn main() {
    let v1: Option<i32> = Some(100);
    let num = v1.unwrap_or_else(default_number);
    println!("{}", num);    // 100

    let v2: Option<i32> = None;
    // 函数可以作为参数
    let num = v2.unwrap_or_else(default_number);
    println!("{}", num);    // 66

    let v3: Option<i32> = None;
    // 用闭包作为函数参数
    let num = v3.unwrap_or_else(|| 88);
    println!("{}", num);    // 88

    // 有参数的闭包
    let test_closure = |num: i32| -> i32 {
        return num + 100;
    };
    println!("{}", test_closure(10)); // 110
}

fn default_number() -> i32 {
    66
} 
```

闭包要实现的三种trait：

FnOnce <--- FnMut <--- Fn

FnMut继承FnOnce，Fn继承FnMut

FnOnce 只能被调用一次的闭包
